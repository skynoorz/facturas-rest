
# Facturas REST

Codigo de ejemplo para un servicio REST usando H2


## Run Locally

Clone the project

```bash
  git clone https://github.com/skynoorz/facturas-rest
```

Go to the project directory

```bash
  cd facturas-rest
```

Building code

```bash
  mvn clean package
```

Start the server

```bash
  java -jar /target/demo-spring-0.0.1-SNAPSHOT.jar
```
